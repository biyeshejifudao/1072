# 宠物在线销售商城

## 项目简介
宠物在线销售商城是基于Java Web技术，Mysql数据库、JSP页面技术的宠物销售商城。包含两个子项目：1.前端商城 2.后台管理系统 前端商城是用户住注册/登录,浏览宠物信息、加入购物车、下单等。适合作为毕业设计、课程设计、大作业。

https://bysj.bsitem.com/#/home/goodDetail?goodId=1072

## 技术介绍
Java Web技术、SSH框架、Mysql数据库、JSP页面技术

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK8

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
